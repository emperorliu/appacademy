### CLI version of the classic Tic Tac Toe game

#### How to play:

```
SYNOPSIS
      play.rb [<player>] [<player>] [<size>]

OPTIONS
      <player>
          can be ai or human
      <size>
          is optional.
```

For example:
```
> ruby play.rb human ai
```
![3x3][3x3]

or

```
> ruby play.rb ai ai 4
```
![4x4][4x4]

If any player is human, they will be prompted to enter their name
